# **App Name**: GridSpace MVP

## Core Features:

- Live Grid Canvas: A high-fidelity interactive canvas demonstrating the ultra-light 48px–64px grid with rgba(0,0,0,0.04) lines.
- Intelligent Copy Tool: An AI-powered tool that reasons about the user's project type to generate minimalist hero headlines and body text.
- Background FX Controller: Fine-tune subtle radial gradients, blurred highlights, and edge fading via minimalist sliders.
- Tailwind Code Export: One-click export of the clean CSS and Tailwind configuration for the background design.
- Layout Toggles: Switch between centered hero layouts and left-aligned Apple-style content distributions.

## Style Guidelines:

- Primary color: Deep Charcoal (#2c2c2e) used for high-contrast typography against light surfaces.
- Background color: Pure off-white (#f7f7f5) for an airy and professional startup aesthetic.
- Accent color: Muted Blue-Gray (#5a6675) to provide professional and understated highlights.
- Headline and Body font: 'Inter' to maintain a modern, machined, and neutral objective look.
- A breathable, high-whitespace layout utilizing a 48px–64px underlying grid system for structural balance.
- Light-weight, thin-stroke monochrome icons to complement the minimalist design.
- Buttery smooth fades and gentle layout shifts that reinforce the premium feeling.
